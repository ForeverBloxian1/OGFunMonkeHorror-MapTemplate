# OG Fun Monke Horror — Map Template

The official map template for creating custom maps for OG Fun Monke Horror on Meta Quest.

## Requirements

- Unity 2022.3 LTS
- Android Build Support module installed in Unity Hub

## Getting Started

1. Download the latest release from the [Releases](../../releases) page
2. Extract the zip and open the project in Unity 2022.3 LTS
3. Open **OG Fun Monke Horror → Guide** for full documentation

## Uploading Your Map

When your map is ready, export it using **OG Fun Monke Horror → Export** and upload the resulting `.zip` to the OG Fun Monke Horror mod page:

**https://mod.io/g/og-fmh**
